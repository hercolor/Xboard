# 4376 APP 账号体系与注册找回任务

## 目标

1. 后端用户系统增加手机号能力，支持邮箱或手机号登录。
2. 客户端增加注册页面并对接 XBoard API。
3. 客户端增加忘记密码页面并对接 XBoard API。
4. 完成必要代码审核、验证与 git 提交。

## 边界

- 保持旧邮箱登录/注册 API 兼容。
- 新增手机号字段时避免破坏现有用户。
- 客户端继续隐藏订阅 URL、auth_data、节点密码等敏感信息。
- 本轮优先实现账号入口闭环；支付/订单/公告/知识库等商业闭环另列后续任务。

## 验收要点

- 后端 `v2_user` 支持手机号唯一字段。
- 登录接口可用邮箱或手机号登录。
- 注册接口可接收手机号字段并返回原有 auth data。
- 忘记密码接口继续支持邮箱验证码；客户端可完整发起邮箱找回。
- 客户端登录页支持邮箱/手机号输入。
- 客户端注册页、忘记密码页具备表单校验、验证码发送、API 调用和错误提示。
- 通过静态/单元级验证，并完成代码审核与提交。
