# 4376 APP 账号找回增强任务

## 目标

1. 忘记密码支持 `account` 参数，兼容邮箱或手机号。
2. 验证码发送接口支持 `account` 参数。
3. 手机号找回不直接引入短信服务；先通过该手机号绑定用户的邮箱发送验证码。
4. 保持旧 `email` 参数兼容，避免破坏已有 Web/API 调用。

## 边界

- 本轮不接入短信供应商，不新增短信费用或密钥配置。
- 验证码仍使用既有邮箱验证码缓存与邮件模板。
- 客户端只展示普通用户能理解的账号找回文案，不展示后端实现细节。

## 验收要点

- `sendEmailVerify` 可接收邮箱或手机号账号。
- `forget` 可接收邮箱或手机号账号并重置对应用户密码。
- 旧客户端继续用 `email` 字段也能工作。
- 完成 PHP 语法/静态检查、客户端 Dart 静态检查和 git 提交。
